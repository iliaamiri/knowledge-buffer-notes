# My quick cheatsheets

## Tmux Sessions

### Rename selected session:
```
# Enter the sessions menu
<Ctrl-b> s

# Highlight the session (using j or k)

# Go into rename current session mode:
<Ctrl-b> $

# Rename and hit enter once done
```
